---
title: "Here's how we created PurpleMarker and how you can too"
date: 2014-02-09
---

1. Learn from our mistakes by following the instructions below
2. Install Drupal 7 via easiest method available, i.e. Softaculous
    1. Mind Drupal's helpful warnings to stop from leaving open security holes
3. Install [CiviCRM module](https://civicrm.org/go/download) in Drupal
    1. create a separate database for CiviCRM specific data.
    2. install dependencies as necessary: [Chaos Tools](https://drupal.org/project/ctools), [Views](https://drupal.org/project/views)
4. Install [Sunlight integration](https://civicrm.org/extensions/sunlight-congressional-districts-optional-civicrm-integration) (Drupal module)
    1. [https://drupal.org/project/cd\_sunlight](https://drupal.org/project/cd_sunlight)
    2. [apply for Sunlight Foundation API key](http://sunlightfoundation.com/api/accounts/register/)
5. [Enable lookup of CiviCRM contacts by Congressional District](http://https://www.advomatic.com/blogs/dave-hansen-lange/quick-and-easy-congressional-district-lookups-your-civicrm-contacts)
6. [Enable CiviCRM Mapping and Geocoding](http://wiki.civicrm.org/confluence/display/CRMDOC/Mapping+and+Geocoding)
7. [Import your Contacts](https://wiki.civicrm.org/confluence/display/CRMDOC/Importing+Data)
8. [Familiarize yourself](http://wiki.civicrm.org/confluence/display/CRMDOC/CiviCRM+Documentation) with the immense capabilities of CiviCRM
